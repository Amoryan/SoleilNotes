## 1 运算符的优先级

|                     |                          |
| ------------------- | ------------------------ |
| 算术运算符 -> 优先1 | *\* 、\*、/、//、%、+、- |
| 位运算符 -> 优先2   | <<（>>）、&、\|          |
| 比较运算符 -> 优先3 | >、<、>=、 <=、==、!=    |
| 布尔运算符 -> 优先4 | and、or                  |
| 赋值运算符 -> 优先5 | =                        |
