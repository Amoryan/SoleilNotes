## 1 数据类型转换

| 函数名  | 作用       | 注意事项                                                     | 举例                     |
| ------- | ---------- | ------------------------------------------------------------ | ------------------------ |
| str()   | 转成字符串 | 也可用引号                                                   | str(123)<br>'123'        |
| int()   | 转成整数   | 1.文字类和小数类字符串无法转成整数<br>2.浮点类转成整数，抹零取整 | int('123')<br>int(9.6)   |
| float() | 转成浮点型 | 1.文字类字符串无法转成浮点型<br>2.整数转成浮点数，末尾为 0   | float('9.9')<br>float(9) |

```python
print(str(123))
print('123')
print(int('123'))
print(int(123.33))
print(float('9.9'))
print(float(9))
'''
输出：
123
123
123
123
9.9
9.0
'''
```
