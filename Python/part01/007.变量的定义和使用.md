## 1 变量

是内存中一个带标签的盒子。

```python
name = 'python'
print(name)
```

## 2 组成

变量由三部分组成。

* 标识：对象所存储的内存地址，可以使用 id(obj) 获取。
* 类型：对象的数据类型，可以使用 type(obj) 获取。
* 值：对象所存储的具体类型，可以使用 print(obj) 将值打印输出。

```python 
name = 'python'
print(name)
print('标识：', id(name))
print('类型：', type(name))
print('值：', name)
'''
输出：
标识： 2074825352112
类型： <class 'str'>
值： python
'''
```
