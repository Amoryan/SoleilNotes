## 1 嵌套 if 使用

```python
from decimal import Decimal
# 嵌套if
answer = str(input('您是会员吗？y/n'))
money = float(input('请输入价格：'))
if answer == 'y':
    if money >= 200:
        money = Decimal(money) * Decimal(0.9)  # 9 折
    else:
        money = Decimal(money) * Decimal(0.8)  # 8 折
else:
    money = float(input('请输入价格：'))
print('您需要付款：', money)
```
