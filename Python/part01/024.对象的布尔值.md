## 1 对象的布尔值

Python 中一切皆为对象，所有对象都有一个布尔值。

- bool()：获取对象的布尔值。
- Flase、0、None、空字符串、空列表、空元组、空字典、空集合：布尔值为 False。

```python
print(bool(False))
print(bool(0))
print(bool(None))
print(bool(''))
print(bool([]))  # 空列表
print(bool(list()))
print(bool(()))  # 空元组
print(bool(tuple()))
print(bool({}))  # 空字典
print(bool(dict()))
print(bool(set()))  # 空集合
```
