## 1 print

输出函数，可以输出数字、字符串、含有运算符的表达式，可以将内容输出到显示器或者文件中。

```py
# 输出数字
print(11)
print(32.9)

# 输出字符串
print("输出字符串")

# 输出表达式
print(2 + 2)

# 输出到文件
# 1.盘符必须存在
# 2.使用 file = fp
fp = open("D:/test.txt", 'a+')  # a+ 如果文件存在在后面追加，不存在则创建
print("输出到文件中", file=fp)
fp.close()

# 不进行换行输出（输出内容在一行当中）
print('hello', 'world', 'python')
```
