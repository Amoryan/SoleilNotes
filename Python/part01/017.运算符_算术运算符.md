## 1 算术运算符

* 标准算术运算符：加（+）、减（-）、乘（*）、除（/）、整除（//）
* 取余运算符：%
* 幂运算符：**

```python
print(1 + 1)
print(1 - 1)
print(2 * 4)
print(24 / 6)
print(23 // 4) # 整除
print(23 % 4)
print(2 ** 4) # 2 的 4 次方
'''
输出：
2
0
8
4.0
5
3
16
'''

print(9 // 4)  # 2
print(-9 // -4)  # 2

print(-9 // 4)  # -3
print(9 // -4)  # -3  一正一负的整除，向下取整

print(9 % -4)  # -3 余数 = 被余数 - 除数 * 商
print(-9 % 4)  # 3
```
