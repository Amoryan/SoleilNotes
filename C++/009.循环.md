## 1 循环

```c++
#include <iostream>
using namespace std;
 
int main (){
   int a = 10;
    // 1.while
    while (a < 20) {
        cout << "a:" << a << endl;
        a++;
    }

    //2.for
    for (int b = 10; b < 20; b++) {
        cout << "b:" << b << endl;
    }
    
    //基于范围的for循环(C++11)
    int my_array[5] = { 1, 2, 3, 4, 5 };
    // 不会改变 my_array 数组中元素的值
    // x 将使用 my_array 数组的副本
    for (int x : my_array) {
        x *= 2;
        cout << x << endl;
    }

    // 会改变 my_array 数组中元素的值
    // 符号 & 表示 x 是一个引用变量，将使用 my_array 数组的原始数据
    // 引用是已定义的变量的别名
    for (int &x : my_array) {
        x *= 2;
        cout << x << endl;
    }

    // 还可直接使用初始化列表
    for (int x : {1, 2, 3, 4, 5}) {
        x *= 2;
        cout << x << endl;
    }
}
```

## 2 循环控制语句

| 控制语句      | 描述                                                         |
| :------------ | :----------------------------------------------------------- |
| break         | 终止 **loop** 或 **switch** 语句，程序流将继续执行紧接着 loop 或 switch 的下一条语句。 |
| continue 语句 | 引起循环跳过主体的剩余部分，立即重新开始测试条件。           |
| goto 语句     | 将控制转移到被标记的语句，不建议在程序中使用 goto 语句。     |
