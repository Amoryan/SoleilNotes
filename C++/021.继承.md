## 1 继承

当创建一个类时，您不需要重新编写新的数据成员和成员函数，只需指定新建的类继承了一个已有的类的成员即可。这个已有的类称为**基类**，新建的类称为**派生类**。

```c++
//基类
class Animal {
    // eat() 函数
    // sleep() 函数
};

//派生类
class Dog : public Animal {
    
};

class Dog : public Animal {
    // bark() 函数
};
//注意：和 Java 不同 c++ 支持 多继承
class Dog:public Animal,public Person {
    
}
```
