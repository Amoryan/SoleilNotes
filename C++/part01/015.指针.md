## 1 指针
指针是一个变量，其值为另一个变量的地址，即内存位置的直接地址。
```c++
int    *ip;    /* 一个整型的指针 */
double *dp;    /* 一个 double 型的指针 */
float  *fp;    /* 一个浮点型的指针 */
char   *ch;    /* 一个字符型的指针 */

```

## 2 使用指针
```c++
#include <iostream>
using namespace std;

int main (){
   int  var = 20;   // 实际变量的声明
   int  *ip;        // 指针变量的声明
   ip = &var;       // 在指针变量中存储 var 的地址
   cout << var << endl;
   
   // 输出在指针变量中存储的地址
   cout << ip << endl; 
 
   // 访问指针中地址的值
   cout << *ip << endl;
 
   return 0;
}

```
