## 1 数组

存储一个固定大小的相同类型元素的顺序集合。

```c++
#include <iostream>
using namespace std;
int main(){
    int a[10];
    for(int i = 0;i < 5;i++){
        a[i] = i; //访问数组
    }    
}

```

## 2 指向数组的指针

```c++
double *p;
double a[10];
p = a;
//*p、*(p + 1) 等访问数组元素
```

## 3 传递数组给函数

```c++
void func(int *p)
void func(int p[10])
void func(int p[])    
```

## 4 函数返回数组

```c++
int * getRandom(){
    static int r[1]; // 局部函数 必须使用 static
    r[1] = 10;
    return r;
}
```
