## 题目描述

大家都知道斐波那契数列，现在要求输入一个整数n，请你输出斐波那契数列的第n项（从0开始，第0项为0，第1项是1）。

> 输入：4
>
> 输出：3

```java
//1.递归法
public class Solution {
    public int Fibonacci(int n) {
        if (n == 0 || n == 1) {
            return n;
        }
        return Fibonacci(n - 1) + Fibonacci(n - 2);
    }
}

//2.优化递归
public class Solution {
    public int Fibonacci(int n) {
        if (n == 0) {
            return 0;
        }
        int a = 1, b = 1;
        for (int i = 1; i <= n - 2; i++) {
            a = a + b;
            b = a - b;
        }
        return a;
    }
}
```
