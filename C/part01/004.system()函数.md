## 1 system()
包含在头文件 <stdlib.h> 中。
```c
#include<stdlib.h>
int system(const char *command);
功能：在已经运行的程序中执行另外一个外部程序
参数：外部可执行程序名字
返回值：
- 成功：0
- 失败：任意数字

//例：
#include <stdio.h>
#include <stdlib.h>

int main()
{
    //system("calc"); //windows平台
	system("ls"); //Linux平台, 需要头文件#include <stdlib.h>
	system("pause"); //暂停
    system("cls"); //清屏
    system("date"); //显示年月日
    system("time"); //显示小时分钟秒
	return 0;
}
```
